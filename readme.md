# MyAssign - AutoHotKey カスタムキーアサイン

AutoHotKey v2.0を使用した個人用キーアサイン定義ファイルです。

## 概要

このスクリプトは、Windows環境での作業効率を向上させるために、各種キーボードショートカットをカスタマイズします。特に、無変換キーやALTキーを活用したカーソル移動、IME制御、アプリケーション固有の機能を提供します。

## 必要要件

- **AutoHotkey v2.0** 以降

## 主な機能

### IME制御

- **変換キー**: IME を ON
- **無変換キー**: IME を OFF
- **ひらがな/カタカナキー**: IME を ON

### カーソル移動（Vim風）

左ALTキーまたは無変換キーと組み合わせて使用します。Shiftキーを併用することで選択も可能です。

| キー | 動作 |
|------|------|
| `ALT + h` / `無変換 + h` | 左カーソル移動 |
| `ALT + j` / `無変換 + j` | 下カーソル移動 |
| `ALT + k` / `無変換 + k` | 上カーソル移動 |
| `ALT + l` / `無変換 + l` | 右カーソル移動 |
| `ALT + u` / `無変換 + u` | PageDown |
| `ALT + i` / `無変換 + i` | PageUp |
| `ALT + y` / `無変換 + y` | Home |
| `ALT + o` / `無変換 + o` | End |

### マウスホイールエミュレーション

無変換キーと組み合わせて、キーボードでスクロール操作が可能です。

| キー | 動作 |
|------|------|
| `無変換 + m` | スクロールダウン |
| `無変換 + ,` | スクロールアップ |
| `無変換 + n` | 左スクロール |
| `無変換 + .` | 右スクロール |

### アプリケーション固有機能

#### 日付入力（Excel以外）

| キー | 動作 |
|------|------|
| `Ctrl + ;` | 日付入力（yyyy/mm/dd形式） |
| `Ctrl + Shift + ;` | 日付入力（yyyymmdd形式） |

#### Excel専用

| キー | 動作 |
|------|------|
| `Ctrl + Shift + V` | 書式なしペースト |

#### PowerPoint専用

| キー | 動作 |
|------|------|
| `Ctrl + 1`（フルキー） | 図形の書式 |

### その他の便利機能

| キー | 動作 |
|------|------|
| `無変換 + v` | バージョン表示 |
| `無変換 + t` | TAB文字をペースト |
| `テンキーのENTER` | フルキーのENTER |
| `Shift + Ctrl + P` | パスワード生成（VSCode/Antigravityを除く） |
| `CapsLock` | 無効化（Shift + CapsLockは使用可能） |

## 使用方法

1. [AutoHotkey v2.0](https://www.autohotkey.com/)をインストール
2. `MyAssign.ahk`をダブルクリックして実行
3. タスクトレイにAutoHotkeyアイコンが表示されれば起動完了

### スタートアップ登録（推奨）

Windows起動時に自動実行するには、以下の手順でスタートアップフォルダに登録します：

1. `Win + R`で「ファイル名を指定して実行」を開く
2. `shell:startup`と入力してEnter
3. 開いたフォルダに`MyAssign.ahk`のショートカットを配置

## 注意事項

- 一部の機能は特定のアプリケーション（Excel、PowerPoint、VSCode、Antigravityなど）で動作が制限されています
- CapsLockキーは無効化されていますが、Shift + CapsLockは使用可能です
- パスワード生成機能はVSCodeとAntigravityでは無効化されています（コマンドパレットとの競合回避）
